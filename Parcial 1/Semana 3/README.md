# Gestión de Mascotas en Python

**Autor:** Jose [Tu Apellido]

Este proyecto presenta dos enfoques para resolver el mismo problema: registrar y mostrar información de mascotas.

## Programación Tradicional
- Uso de funciones y variables.
- Entrada de datos por teclado.
- Ejemplo en `programacion_tradicional/tradicional.py`.

## Programación Orientada a Objetos
- Uso de clases, objetos, atributos y métodos.
- Ejemplo en `programacion_orientada_objetos/mascota.py` y `main.py`.

## Reflexión
La programación tradicional es más directa y sencilla, pero se vuelve difícil de escalar cuando el sistema crece.  
La programación orientada a objetos permite organizar mejor el código, reutilizarlo y trabajar con abstracciones que representan entidades del mundo real.
